# Docker & Kubernetes para Desarrolladores

Curso práctico de 20 horas para aprender a construir, ejecutar, asegurar y desplegar aplicaciones con Docker, Docker Compose y Kubernetes.

## Al finalizar el curso podrás

- Construir imágenes Docker desde un `Dockerfile`.
- Ejecutar y depurar contenedores.
- Orquestar servicios con Docker Compose.
- Aplicar escaneo básico de imágenes.
- Desplegar cargas de trabajo en Kubernetes.
- Exponer servicios y gestionar configuración.
- Aplicar probes y escalado horizontal básico.

## Formato

- 10 sesiones de 2 horas.
- Cada sesión tiene conceptos, laboratorio guiado y validación.
- Modalidad práctica.
